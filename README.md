# Sushanth Kathirvelu Portfolio

Static portfolio site ready for GitHub Pages or any static host.

## Files

- `sushanth-kathirvelu-portfolio.html` — main homepage
- `sushanth-kathirvelu-projects.html` — project case studies page
- `assets/Sushanth_Kathirvelu_Resume.pdf` — downloadable resume

## Quick GitHub hosting

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Rename `sushanth-kathirvelu-portfolio.html` to `index.html` if you want it to open as the site root on GitHub Pages.
4. In GitHub, open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select your main branch and `/root` folder.
7. Save, then wait for GitHub Pages to publish.
